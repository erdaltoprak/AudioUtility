# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog 1.1.0](https://keepachangelog.com/en/1.1.0/),  
and this project aims to adhere to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0]

### Added

- Initial public release of AudioUtility.
- Menu bar interface for switching the system’s default input and output devices without opening System Settings.
- Automatic device refresh and CoreAudio observation for device list and default-device changes.
- Inline edit mode for fallback ordering, device exclusion, and remembered disconnected devices.
- Onboarding flow with launch behavior guidance and Open at Login setup.
- Loading, retry, and empty-state handling for device discovery and switching.
- Menu bar settings, about links, and Sparkle update support for direct releases.
